  • Q. What is GitHub?
  • A. GitHub is a user-friendly company that is a cloud based Git repository hosting service. GitHub allows individuals or companies to easily work on a wide variety of projects together. Source: https://kinsta.com/knowledgebase/what-is-github/
  • Q. What is the difference between Git and GitHub?
  • A. The main differences that distinguish GitHub from Git is that GitHub is a repository hosting service, and Git is the software. GitHub is cloud based and is for-profit (even though using and making an account is free). Source: https://devmountain.com/blog/git-vs-github-whats-the-difference/
  • Q. Who started GitHub and how was it started?
  • A. GitHub was started by Tom Preston-Werner, Chris Wanstrath, and PJ Hyett in 2008. It was created with the intention of having a service that allowed others to easily collaborate on and manage code on projects together. Source: https://nira.com/github-history/
  • Q. What company owns it now?
  • A. Microsoft, aquired in 2018. Source: https://nira.com/github-history/
  • Q. How much does a GitHub account cost?
  • A. A GitHub account is free to create and use, however, you can purchase upgrades for a monthly fee depending on the needs of your project/company. Source: https://github.com/pricing
