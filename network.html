Chapter 4 Terminology

Term Definition
Networking Classifications
    1.LAN
    2.MAN
    3.WAN
    1. a local area network, consists of a collection of computers in a single building or building complex; ex: the computers on a college campus.
    2. a metropolitan area network. A network of intermediate size, such as spanning a local community.
    3. a wide area network. The largest spanning, links machines over neighboring cities and beyond.
Network Models
    1.Bus Network
    2.Star Network
    1. One of the patterns in which the machines are connected; in bus network, they are connected to a common communication line called a bus
    2. one machine serves as a central focal point to which all the others are connected.
Protocols
    1.CSMA/CD Ethernet
    2.CSMA/CA Wifi
    1. carrier sense, multiple access with collision detection; this protocol dictates that each message be broadcast to all the machines on the bus.
    2. carrier sense, multiple access with collision avoidance; the policy of trying to avoid colllisions rather than trying to detect them.
Combining Networks
    1.Repeater
    2.Bridge
    3.Switch
    4.Router
    5.Gateway
    1. a device that passes signals back and forth between the two original buses without considering the meaning of the signals.
    2. more complex than a repeater; it connects two buses, but it doesn't pass all messages across the connection. It looks at the destination address that accompanies each message across the connection only when that message is destined for a computer on the other side.
    3. a bridge with multiple connections, allowing it to connect several buses rather than just two. So, a switch produces a network consisting of several buses extending from the switch, as spokes on a wheel.
    4. the connection between networks to form an internet; special purpose computers used for forwarding messages.
    5. the point at which one network is linked to an internet; it serves as a passageway between the network and the outside world.
1.Client/Server Model
    2. p2p
    3.Distributed System
    4.Cloud Computing
    1. a popular convention used for interprocess communication.
    2. peer-to-peer:involves processes that provide service to and receive service from each other. ex: instant messaging
    3. consists of software units that execute as processes on different computers.
    4. huge pools of shared computers on the network can be allocated for use by clients as needed.
Internet Addressing
    1.Tier 1
    2.Tier 2
    3.Tier 3
    1. ISPs that consist of very high-speed, high-capacity, international WANs;"the backbone of the internet".
    2. more regional in scope than tier 1 and less potent in their capabilities.
    3. access to the core is provided by this tier.
1.IP Address
    2.ICANN
    3.DNS
    4.TLD
    5.Country Code TLDs
    1. an internet-wide addressing system that assigns a unique identifying address to each computer in the system.
    2. a nonprofit corporation established to coordinate the internet's operation
    3. domain name system; name servers that are used as an internet wide directory system.
    4. Top-level domain:suffixes after the domain name of a site. ex: .com, .org, etc.
    5. two letter TLDs such as .au or .ca to include other countries.
Internet Protocols
    1.FTP
    2.HTTP
    3.SMTP
    1. an application for listing and copying files across the network.
    2. hyper text transfer protocol; hypertext documents that are normally transferred between browsers and websavers.
    3. simple mail transfer protocol; a way that two computers on the network may interact when transmitting an emailmessage from one host to the other.
World Wide Web
    1.URL
    2.Client-side Activities
    3.Server-side Activities
    1. uniform resource locator; in order to locate and retrieve documents on the web, each document is given a unique address.
    2. activities performed on a webpage by a client (such as a browser)
    3. activities performed by a server (such as a webserver)
Internet Four-Layer Software
    1.Application Layer
    2.Transport Layer
    3.Network Layer
    4.Link Layer
    1. consists of those software units such as clients and servers that use internet communication to carry out their tasks.
    2. accepts messages from the application layer and ensure that the messages are formatted properly for transmission over the internet one segment at a time.
    3. decides in which direction a packet should be sent at each step along the packet's path through the internet.
    4. in charge of receiving and transmitting the packets.
Internet is:
    1.Packet-Switching network
    2.Fault Tolerant
    3.Redundant
    1. digital data is sent by breaking the data into blocks of bits called packets, which contain both the data being transmitted and control information for routing the data.
    2. able to keep everything running smoothly
    3. more than one way to route data.
Cybersecurity
    1.Malware .
    2.Virus
    3.Worm
    4.Trojan Horse
    5.Spyware
    6.Phishing
    7.DDoS Attack
    1. malicious software that might be transferred to, and executed on, the computer itself, or it might attack the computer from a distance
    2. a software that infects a computer by inserting itself into programs that already reside in the machine; when the host program is executed, so is the virus.
    3. an autonomous program that transfers itself through a network, taking up residence in computers and forwarding copies of itself to other computers.
    4. a program that enters a computer system disguised as a desireable program, such as a game or useful utility package, that is willingly imported by the victim. Once in the computer, the trojan horse performs additional activities that might have harmful effects.
    5. software that collects information about activities at the computer on which it resides and reports that information back to the instigator of the attack.
    6. a technique of obtaining information by simply asking for it.
    7. the process of overloading a computer with messages distributed over many hosts.
Protection
    1.Firewall
    2.Spam Filter
    3.Proxy Server
    4.Antivirus Software
    5.Cryptography
    6.Public Key Encryption
    1. installed at the gateway of an organization's intranet to filter messages passing in and out of the region.
    2. firewalls designed to block unwanted email.
    3. a software that acts as an intermediary between a client and a server with the goal of shielding the client from adverse actions of the server.
    4. used to detect and remove the presence of known viruses and other infections.
    5. the study of sending and receiving messages securely in the presence of adversaries.
    6. used to encrypt messages.
